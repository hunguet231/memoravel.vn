# Memoravel.vn

## Run client

### Install packages

```
  cd client
  npm install
```

### Run server

```
  npm run dev
```

## Run server

### Install packages

```
  cd server
  npm install
```

### Config env

- Create new file .env in folder server
- Copy file .env.example to file .env

### Run server

```
  npm start
```
